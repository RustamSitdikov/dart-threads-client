Creds.fromStrings(String DBID, { String pubKey String signature })

A helper function if you have String representations of the required params.

Implementation

Creds.fromStrings(String DBID, {String pubKey, String signature}) {
  this.DBID = utf8.encode(DBID);
  if (pubKey != null) {
    this.pubKey = utf8.encode(pubKey);
  }
  if (signature != null) {
    this.signature = utf8.encode(signature);
  }
}
